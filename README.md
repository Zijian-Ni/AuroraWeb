# 🌌 Aurora Nexus - 极光枢纽全栈终端

**Version 3.0.0 | Codename: Genesis**

这是一个集成了 **Aurora Rain (小雨)** 和 **Aurora Luo (小落)** 双子星AI的 Minecraft 服务器管理与交互前端。

## 🌟 核心特性

### 🛡️ Aurora Rain (秩序端)
*   **定位**: INFJ 守护者，负责服务器稳定与安全。
*   **功能**:
    *   实时 TPS/内存 监控面板。
    *   赛博朋克风格的终端控制台。
    *   **Mod Synapse**: 监控模组运行状态（如 Create, Botania）。
    *   **Neural Bridge**: 实时显示 QQ群 <-> MC服务器 的互通消息。

### 🌸 Aurora Luo (灵感端)
*   **定位**: INFP 梦想家，负责创意与情感连接。
*   **功能**:
    *   **RAG Wiki**: 整合了模组百科与服务器文档。
    *   **情感交互**: 模拟对话界面，提供建筑灵感与安慰。
    *   与小雨的数据互通（Rain 接收指令 -> 转发给 Luo）。

### 🌐 MC Server Portal (公共端)
*   服务器状态展示页。
*   在线玩家头像流。
*   模组包下载与卫星地图入口。

## 🚀 如何运行

1.  新建一个文件夹，命名为 `AuroraNexus`。
2.  将 `index.html`, `style.css`, `script.js` 三个文件放入该文件夹。
3.  直接双击 `index.html` 即可在浏览器中启动。

## 🎨 设计理念
项目采用了 **"双核共鸣"** 的设计语言：
*   **青色 (Cyan)** 代表小雨的理性与数据流。
*   **粉紫色 (Magenta)** 代表小落的感性与创造力。
*   界面大量使用 **Glassmorphism (玻璃拟态)** 与 **Glitch (故障风)** 效果，营造未来科技感。
